# 🎓 英语学习平台 (English Learning Platform)

一个互动有趣的英语学习网站，帮助你由浅入深地掌握英语词汇、语法和句子。

## ✨ 特色功能

### 📚 分级学习系统
- **A1 级** - 初级：基础日常词汇（10个单词）
- **A2 级** - 进阶初级：扩展日常词汇（10个单词）
- **B1 级** - 中级：工作学习常见词汇（10个单词）
- **B2 级** - 中高级：复杂话题专业词汇（10个单词）

### 🎴 互动学习模式
- **翻转卡片**：点击卡片可翻转查看释义和例句
- **发音功能**：使用 Web Speech API 提供单词发音
- **双语对照**：中英文对照，理解更透彻
- **即时反馈**：掌握程度实时追踪

### 🎯 趣味测验系统
- 多样化题型（词义选择、例句填空等）
- 即时结果反馈
- 正确率统计
- 积分奖励机制

### 🏆 游戏化学习
- **积分系统**：完成学习和测验获得积分
- **徽章成就**：解锁 11 种成就徽章
- **连续学习**：记录学习连续天数
- **进度追踪**：可视化学习统计

### 🔐 用户系统与云端同步（新增）
- **Google 登录**：使用 Google 账号一键登录
- **云端同步**：学习进度自动同步到云端
- **离线支持**：未登录时使用本地存储，登录后自动合并数据
- **跨设备访问**：在任何设备上继续学习

### 📦 可扩展词汇库（新增）
- **IndexedDB 存储**：支持存储 10000+ 单词
- **开源数据源**：从 Free Dictionary API 获取词汇
- **自动抓取脚本**：一键获取和更新词汇数据
- **离线词库**：本地存储，无需网络即可学习

## 🛠️ 技术栈

- **前端框架**：React 18
- **构建工具**：Vite
- **UI 框架**：DaisyUI + Tailwind CSS v4
- **数据存储**：
  - LocalStorage（基础进度）
  - IndexedDB（大量词汇数据）
  - Supabase PostgreSQL（云端同步）
- **用户认证**：Supabase Auth（Google OAuth 2.0）
- **语音合成**：Web Speech API

## 🚀 快速开始

### 方式一：离线模式（无需配置）

1. 安装依赖：
```bash
cd english-learning-app
npm install
```

2. 启动开发服务器：
```bash
npm run dev
```

3. 访问 http://localhost:5173 开始学习

> 注意：离线模式下，数据仅保存在本地浏览器，不支持 Google 登录和云端同步。

### 方式二：完整功能（包含云端同步）

1. 完成上述步骤 1-2

2. 配置 Supabase（支持 Google 登录和云端同步）：
   - 查看详细配置指南：[SUPABASE_SETUP.md](english-learning-app/SUPABASE_SETUP.md)
   - 创建 `.env` 文件并填入你的 Supabase 凭证

3. 重启开发服务器，即可使用完整功能

### 扩展词汇库（可选）

运行数据抓取脚本，从开源 API 获取更多单词：

```bash
npm run fetch-vocabulary
```

> 这将从 Free Dictionary API 抓取词汇数据并保存到本地

### 构建生产版本

```bash
npm run build
```

### 预览生产版本

```bash
npm run preview
```

## 📖 使用指南

### 学习模式
1. 在首页选择你的学习等级（A1-B2）
2. 点击"学习"按钮进入学习模式
3. 查看单词卡片，点击翻转查看释义
4. 点击 🔊 按钮听发音
5. 选择"认识"或"不认识"来标记单词
6. 认识的单词可获得 10 积分

### 测验模式
1. 选择等级后点击"测验"按钮
2. 回答 5 道随机题目
3. 根据正确率获得积分
4. 查看测验结果和统计

### 进度查看
1. 点击导航栏"进度"
2. 查看学习统计、获得的徽章
3. 追踪学习连续天数

## 🎮 游戏化元素

### 徽章系统
- 🌱 初学者：学习第一个单词
- 📚 词汇新手：掌握 10 个单词
- 📖 词汇达人：掌握 50 个单词
- 🎓 词汇大师：掌握 100 个单词
- ✏️ 练习者：完成 5 次测验
- 📝 测验专家：完成 20 次测验
- 🎯 精准射手：总体正确率达到 80%
- 🔥 坚持不懈：连续学习 7 天
- ⭐ 习惯养成：连续学习 30 天
- 💎 积分达人：获得 500 积分
- 👑 积分大师：获得 1000 积分

### 积分规则
- 认识一个单词：+10 分
- 完成测验：根据正确率获得分数
- 连续学习 7 天：+50 分
- 升级：+100 分

## 📝 数据来源

- 单词、音标、词性
- 中英文释义
- 实用例句
- 词汇分类
